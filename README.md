## Задание 2.2: Тесты UI

### Инструкция по запуску автотестов

- Для автоматизации тест-кейсов задания 2.2 я написала UI-тесты для проверки функциональности сайта модерации. Тесты написаны на **Java** с использованием **Selenium WebDriver** и **JUnit 5**
- Тесты лежат в в файле TESTCASES.md, баг-репорты по найденным во время решения ошибкам лежат в файле BUGS.md


###  Что потребуется для запуска

1. **Google Chrome**
2. **Java Development Kit (JDK)** версии 11, 17 или выше
3. **IntelliJ IDEA** 



### Инструкция по запуску

#### Шаг 1. Открытие проекта в IDE
1. Откройте IntelliJ IDEA
2. На стартовом экране нажмите **Open** (или `File` -> `Open` в верхнем меню)
3. Выберите папку с этим проектом (ту самую папку, внутри которой лежит файл `pom.xml`) и нажмите **OK**
4. Если IDEA спросит *"Trust and Open Project '...'"* — нажмите **Trust Project**

#### Шаг 2. Загрузка зависимостей (Maven)
Проекту нужно скачать библиотеки (Selenium и JUnit) из интернета. Обычно IDEA делает это автоматически, но лучше проверить:
1. Откройте файл `pom.xml` (он находится в корне проекта слева в панели файлов)
2. Если в правом верхнем углу редактора появилась плавающая иконка с синей буквой **"m"** и стрелочками (Load Maven Changes) — **нажмите на нее**
3. Дождитесь окончания загрузки (внизу окна IDEA пробежит статус-бар). Как только красный текст в файле `pom.xml` исчезнет — проект готов к работе

#### Шаг 3. Запуск тестов
1. В левой панели файлов (Project) раскройте папки по следующему пути:  
   `src` -> `test` -> `java`
2. Внутри вы найдете файлы с тестами (например, `ThemeToggleTest`). Дважды кликните по файлу, чтобы открыть его код
3. Слева от номеров строк вы увидите **зеленые треугольники** (кнопки Play)
   * Чтобы запустить **все тесты в файле**, нажмите на верхний зеленый треугольник (рядом со словом `public class...`)
   * Выберите в выпадающем меню **Run**

#### Шаг 4. Наблюдение за результатом
1. IDEA скомпилирует проект.
2. На экране автоматически откроется браузер Google Chrome, робот выполнит все необходимые клики и проверки (например, переключит тему), после чего браузер самостоятельно закроется
3. В нижней части окна IDEA откроется панель результатов (Run):
   *  **Зеленая галочка** означает, что тест пройден успешно, багов не найдено
   *  **Красный крестик** означает, что тест упал. В консоли будет выведено сообщение о том, что именно пошло не так
